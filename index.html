About This Data

Each House committee is required to submit a monthly spending report to the Committee on House Administration. These reports are published only as PDFs, making analysis difficult.

This dashboard automatically extracts financial data from those PDFs and presents it in a searchable, sortable format.

Data Sources & Methodology

  • Source: PDF reports from cha.house.gov/committee-reports
  • Extraction: Text-based PDFs parsed with pdfplumber; scanned/image PDFs read via AI vision (Claude Haiku). 274 of 281 reports (98%) successfully extracted.
  • Scope: 119th Congress (January 2025 – present), 22 standing and select committees
  • Data collected: March 18, 2026

What the Numbers Mean

  • MTD (Month-to-Date): Amount disbursed during that calendar month
  • YTD (Year-to-Date): Cumulative disbursements for the legislative year (resets each January for a new session)
  • Personnel Compensation: Staff salaries and benefits (Object Class 11)
  • Travel: Airfare, lodging, per diem, field hearings (Object Class 21)
  • Rent/Comm/Utilities: Telecom, postage, equipment rental (Object Class 23)
  • Printing & Reproduction: Non-franked printing, photography (Object Class 24)
  • Other Services: Technology contracts, web hosting (Object Class 25)
  • Supplies & Materials: Office supplies, food, software, publications (Object Class 26)
  • Equipment: Purchases and maintenance under $25,000 (Object Class 31)

Filing Status

  • "Not filed": No PDF report found on the CHA website for that committee/month
  • "$0 PDF": A report was filed but the automated extraction could not read the financial table (usually due to non-standard formatting). The original PDF is linked.

Known Limitations

  • AI-assisted OCR may produce minor numerical inaccuracies on scanned documents. Always verify critical figures against the source PDF.
  • Employee count data is incomplete — many scanned reports could not have payroll sections reliably extracted.
  • This is a static snapshot. Data does not update automatically when new reports are posted.
  • Some committees use non-standard report formats that may cause extraction errors.
